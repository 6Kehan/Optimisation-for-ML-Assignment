**Q1：线性回归优化** 

考虑线性回归 $\hat{y}=X\theta$，其中 $\theta\in\mathbb{R}^{2}$，损失函数为 $L(\theta)=\frac{1}{2m}||X\theta-y||_{2}^{2}$ 。
**生成合成数据**：$m=1000$，$X_{ij}\sim\mathcal{N}(0,1)$，$\theta^{*}=[3,4]^{T}$，$y=X\theta^{*}+\epsilon$，噪声 $\epsilon\sim\mathcal{N}(0, \sigma^{2}I)$，其中 $\sigma=1$。初始值设定为 $\theta_{0}=[1,1]^{T}$ 。

* 
**(a) 全批量梯度下降 (Full-batch GD)**：运行 80 次迭代，使用恒定步长（如 $\alpha=0.5$） 。


* 
**绘图**：(i) $L(\theta_{t})$ 随迭代次数变化的曲线（对数坐标）；(ii) 在 $L(\theta_{1},\theta_{2})$ 等高线图上的轨迹 。




* 
**(b) 不同批量大小的小批量 SGD 比较**：每个 epoch 随机打乱一次数据。运行 400 次更新（步长同上） 。


* 比较批量大小 $b=5$ 与 $b=20$ 。


* 
**绘图**：(i) 包含 GD、$SGD(b=5)$ 和 $SGD(b=20)$ 的损失对比图；(ii) 包含这三种算法轨迹的等高线图 。




* 
**(c) SGD 噪声处理 (NAG vs Adagrad)**：使用 $b=10$ 运行 400 次更新，实现 NAG ($\beta=0.9$) 和 Adagrad 算法 。


* 
**绘图**：(i) 比较恒定步长 SGD、NAG 和 Adagrad 的损失图；(ii) 带有轨迹的等高线图 。

**Q2：玩具神经网络模型** 

考虑模型：$\hat{y}(u;x)=x_{2}\tanh(x_{1}u)$，其中 $x=[x_{1},x_{2}]^{T}$ 。
**生成数据**：$m=1000$，$u \sim \text{Uniform}[-2,2]$，$x^{*}=[1,3]^{T}$，$y^{(i)}=\hat{y}(u^{(i)};x^{*})+\epsilon^{(i)}$，其中 $\epsilon^{(i)}\sim\mathcal{N}(0,0.05^{2})$。初始值 $x_{0}=[1,1]^{T}$ 。
损失函数：$J(x)=\frac{1}{m}\sum_{i=1}^{m}\frac{1}{2}(\hat{y}(u^{(i)};x)-y^{(i)})^{2}$ 。

* 
**(a) 全批量 GD 基准**：运行 500 次迭代，恒定步长（如 $\alpha=0.75$） 。


* 
**绘图**：(i) $J(x_{t})$ 对数损失图；(ii) $J(x_{1},x_{2})$ 等高线轨迹图 。




* 
**(b) 小批量 SGD 批量大小比较**：每个 epoch 打乱一次，运行 500 次更新，比较 $b=5$ 和 $b=20$ 。


* 
**绘图**：包含 GD 和两种 SGD 的损失对比图及等高线轨迹图 。




* 
**(c) SGD 噪声处理 (NAG vs Adagrad)**：使用 $b=10$ 运行 500 次更新 。


* 
**绘图**：对比 SGD、NAG 和 Adagrad 的损失图及等高线轨迹图 。

**Q3：Rosenbrock 函数与牛顿法** 

考虑 Rosenbrock 函数：$f(x_{1},x_{2})=(1-x_{1})^{2}+100(x_{2}-x_{1}^{2})^{2}$，起始点 $x_{0}=[-1,1]^{T}$ 。

* 
**(a) 梯度下降基准**：运行 200 次迭代，步长 $\alpha=10^{-3}$。绘制 $f(x_{t})$ 对数损失图 。


* 
**(b) 牛顿法**：使用**有限差分法**计算梯度和海森矩阵 (Hessian)，运行 200 次迭代，恒定缩放因子（如 $\alpha=0.7$）。绘制 $f(x_{t})$ 对数损失图 。


* 
**(c) 阻尼牛顿法 (线搜索阻尼)** ：


* 如果目标函数未下降，则减小步长（例如：收缩因子 0.5，直到函数值下降） 。


* 
**绘图**：(I) GD、牛顿法、阻尼牛顿法的对数损失对比图；(II) 包含 GD（子采样显示）、牛顿法和阻尼牛顿法轨迹的等高线图 。





**提示：阻尼牛顿法（回溯法）步骤简述** ：

1. 计算当前点的函数值、梯度 $g_{t}$ 和海森矩阵 $H_{t}$ 。


2. 求解牛顿方向：$(H_{t}+\lambda I)p_{t}=g_{t}$ 。


3. 尝试步长 $\alpha$。若 $f(x_{new}) < f_{t}$ 则接受；否则将 $\alpha$ 乘以收缩因子 $\rho$ 。


4. 若无接受步长，使用备选方案：$x_{new}=x^{(t)}-10^{-4}g_{t}$ 。
