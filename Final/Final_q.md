# 优化算法综合实验报告
## 一、实验基准问题定义
### 基准A：线性回归（二次损失）
- 样本规模：$m=1000$ 个样本，2维输入特征 $x^{(i)} \in \mathbb{R}^2$
- 真实参数：$\theta^{*}=\begin{bmatrix}3\\ 4\end{bmatrix}$
- 数据生成：$y^{(i)}=\theta^{*T}x^{(i)}+\epsilon^{(i)}$，$\epsilon^{(i)}$ 为零均值高斯噪声
- 损失函数：$J(\theta)=\frac{1}{2m}\sum_{i=1}^{m}(\theta^{T}x^{(i)}-y^{(i)})^{2}$

### 基准B：玩具神经网络（二维非线性函数）
目标函数：
$$f(x_{1},x_{2})=(x_{1}-1)^{2}+5(x_{2}-2)^{2}+\sin(x_{1})$$

### 基准C：Rosenbrock函数（非凸测试函数）
目标函数：
$$f(x_{1},x_{2})=(1-x_{1})^{2}+100(x_{2}-x_{1}^{2})^{2}$$

### 基准D：约束可行集
$$X = \{x \in \mathbb{R}^2: 0.5 \le x_1 \le 5,\ -5 \le x_2 \le 10\}$$

## 二、问题1：Polyak步长、Adagrad、RMSprop、Heavy Ball 优化
### （一）算法实现
1. **Polyak步长梯度下降**
   步长公式：$\alpha_{k}=\frac{f(x_{k})-f^{*}}{||\nabla f(x_{k})||^{2}+\epsilon}$
   - 参数：基准A/B $f^*=0,\epsilon=10^{-4}$；基准C $\epsilon=10^{-3}$，迭代120次
2. **Adagrad（逐坐标自适应步长）**
   - 初始学习率：基准A=1.8，基准B=1.2，基准C=0.45，$\epsilon=10^{-5}$，迭代120次
3. **RMSprop（指数加权平方梯度）**
   - 参数：$\beta=0.9,\epsilon=10^{-5}$；初始学习率：A=0.22，B=0.14，C=0.0035，迭代120次
4. **Polyak动量（Heavy Ball）**
   更新规则：
   $$z_{k+1}=\beta z_{k}+\alpha\nabla f(x_{k}),\ x_{k+1}=x_{k}-z_{k+1}$$
   - 参数：A($\alpha=0.045,\beta=0.88$)、B($\alpha=0.035,\beta=0.90$)、C($\alpha=0.0008,\beta=0.86$)，迭代120次
5. **固定步长GD（基准）**
   学习率：A=0.08，B=0.06，C=0.0012

### （二）实验任务
1. **任务(a)**：完成上述4种优化算法的代码实现
2. **任务(b)**：绘制基准A/B/C的**目标值-迭代次数**收敛曲线
3. **任务(c)**：绘制基准B/C的优化轨迹**等高线图**
4. **任务(d)**：绘制Polyak、Adagrad、RMSprop的**自适应步长演化曲线**
5. **任务(e)**：从**收敛速度、收敛平滑度、算法稳定性**三个维度对比所有算法

## 三、问题2：Nesterov动量、Adam、小批量SGD
### （一）算法实现
1. **Nesterov加速梯度**
   更新规则：
   $$z_{k+1}=\beta_{k}z_{k}-\alpha\nabla f(x_{k}+\beta_{k}z_{k}),\ x_{k+1}=x_{k}+z_{k+1}$$
   $$\beta_{k}=\min\left(\frac{k-1}{k+2},\beta_{max}\right)$$
   - 参数：A($\alpha=0.06,\beta_{max}=0.90$)、B($\alpha=0.035,\beta_{max}=0.92$)、C($\alpha=0.0007,\beta_{max}=0.90$)，迭代150次
2. **Adam（一阶+二阶矩优化）**
   - 参数：$\epsilon=10^{-8}$；A($\alpha=0.12,\beta_1=0.82,\beta_2=0.999$)、B($\alpha=0.08,\beta_1=0.80,\beta_2=0.999$)、C($\alpha=0.006,\beta_1=0.80,\beta_2=0.999$)，迭代150次
3. **小批量SGD（仅基准A）**
   - 参数：$\alpha=0.06$，批量大小 $b=5$ / $b=40$，epoch=50，每轮打乱数据
4. **带噪声SGD（仅基准A）**
   - 参数：标签噪声放大6倍，其余参数与小批量SGD一致
5. **固定步长GD（基准）**
   学习率：A=0.08，B=0.06，C=0.0012

### （二）实验任务
1. **任务(a)**：在基准A/B/C上对比Nesterov、Adam、固定步长GD性能
2. **任务(b)**：绘制**目标值-迭代次数**收敛曲线
3. **任务(c)**：绘制基准B/C的优化轨迹**等高线图**
4. **任务(d)**：分析SGD中**批量大小**对收敛的影响
5. **任务(e)**：分析**额外噪声**对SGD收敛性的改变

## 四、问题3：牛顿法与局部近似
### （一）一维泰勒近似
测试函数：$g(x)=x^4$，初始点 $x_0=0.25$
1. **一阶近似**：$g(x) \approx g(x_0)+g'(x_0)(x-x_0)$
2. **二阶近似**：$g(x)\approx g(x_{0})+g^{\prime}(x_{0})(x-x_{0})+\frac{1}{2}g^{\prime\prime}(x_{0})(x-x_{0})^{2}$
3. **一维牛顿更新**：$x_{k+1}=x_k - \frac{g'(x_k)}{g''(x_k)}$

### （二）牛顿法实现
- 阻尼系数：$10^{-8}$
- 学习率：基准A=1.0，基准B=0.85，基准C=0.22
- 迭代次数：GD=80次，牛顿法=20次
- 梯度下降基准：A=0.08，B=0.06，C=0.001

### （三）实验任务
1. **任务(a)**：绘制原始函数$x^4$、一阶近似、二阶近似对比图
2. **任务(b)**：在基准A/B/C上实现牛顿法
3. **任务(c)**：对比牛顿法与梯度下降法
4. **任务(d)**：绘制基准B/C的优化轨迹**等高线图**
5. **任务(e)**：绘制GD与牛顿法**更新幅度-迭代次数**曲线

## 五、问题4：导数近似与无导数优化
### A部分：基准B（有限差分 + Nesterov随机搜索）
1. **前向有限差分梯度**
   公式：$\frac{\partial f}{\partial x_{i}}(x)\approx\frac{f(x+\delta e_{i})-f(x)}{\delta}$
   - 良好设置：$\alpha=0.08,\delta=0.05$；不良设置：$\alpha=0.08,\delta=0.8$，迭代120次
2. **Nesterov随机搜索**
   公式：$x_{k+1}=x_{k}-\alpha\frac{f(x_{k}+\delta u_{k})-f(x_{k})}{\delta}u_{k}$
   - 参数：$\alpha=0.025,\delta=0.08$，迭代220次
3. **基准**：精确梯度下降，$\alpha=0.06$

### B部分：基准C（Nelder-Mead + 网格搜索）
1. **Nelder-Mead单纯形法**
   - 参数：初始步长0.35，迭代160次
2. **网格搜索**
   - 搜索范围：$x_1\in[-2,2],x_2\in[-1,3]$，网格尺寸$55\times55$

### 实验任务
1. **任务(a)**：基准B上对比精确GD、有限差分GD、Nesterov随机搜索
2. **任务(b)**：绘制**目标值-迭代次数**曲线
3. **任务(c)**：绘制基准B优化轨迹**等高线图**
4. **任务(d)**：绘制基准C的Nelder-Mead优化轨迹
5. **任务(e)**：绘制基准C网格搜索采样点与最优值

## 六、问题5：约束优化（投影GD + 惩罚法）
约束条件：基准B + $x_1 \ge 0.5$，初始点：$(0.2, 4.0)$（不可行点）
### （一）算法实现
1. **投影梯度下降**
   投影规则：$x_{1}\leftarrow \max(0.5,x_{1})$
   - 参数：$\alpha=0.08$，迭代100次
2. **惩罚法**
   惩罚目标：$F_{\lambda}(x)=f(x)+\lambda \max(0,-x_{1}+0.5)$
   - 参数：$\lambda=0.15/1.8/4.5$，$\lambda=4.5$时$\alpha=0.03$，迭代100次
3. **无约束GD**
   - 参数：$\alpha=0.07$

## 七、问题6：线性规划与Frank-Wolfe算法
可行集：$\mathcal{X}=\{x\in\mathbb{R}^{2}:0.5\le x_{1}\le5,-5\le x_{2}\le10\}$
### （一）算法实现
1. **线性规划**
   目标函数：$f(x)=a^{T}x,\ a=\begin{bmatrix}1\\ 2\end{bmatrix}$，求解可行域上的最优解
2. **Frank-Wolfe（内部最优解）**
   目标函数：$f(x_{1},x_{2})=(x_{1}-1)^{2}+(x_{2}-5)^{2}$
   更新规则：
   $$z_{k}=\arg\min_{x\in\mathcal{X}}\nabla f(x_{k})^{T}x,\ x_{k+1}=\beta x_{k}+(1-\beta)z_{k}$$
   - 参数：$x_0=(1,1)$，$\beta=0.90/0.985$，迭代180次
3. **Frank-Wolfe（边界最优解）**
   目标函数：$f(x_{1},x_{2})=x_{1}^{2}+x_{2}^{2}$
   - 参数：$x_0=(3,3)$，$\beta=0.93$，迭代140次

### 实验任务
1. 绘制可行域与线性目标函数**等高线图**
2. 绘制内部最优解下Frank-Wolfe**收敛曲线**
3. 绘制Frank-Wolfe优化轨迹**等高线图**
4. 绘制$x_k$和$z_k$**迭代演化曲线**
5. 对比最优解在**内部/边界**时的算法行为差异
